Cue Gallery Favicon Example

This example demonstrates how to fetch and use images from the Cue gallery as favicons.

Preview:

How it works

  1. Click "Update Favicon" to fetch the latest image from Cue gallery
  2. The favicon will change to a random variation from the latest generation
  3. Look at the browser tab to see the new favicon

Integration Code

// Fetch and update favicon
async function updateFaviconFromCue(apiUrl) {
    const response = await fetch(`${apiUrl}/gallery/latest`);
    const data = await response.json();
    
    if (data.image_data_url) {
        // Update favicon
        const link = document.querySelector("link[rel*='icon']") || 
                    document.createElement('link');
        link.type = 'image/png';
        link.rel = 'icon';
        link.href = data.image_data_url;
        document.head.appendChild(link);
    }
}
